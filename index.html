About

This directory is fully for 3.0.0 scripts.js development. There are some very useful (sometimes buggy) scripts in here that you can use for your own scripts!

Batch

Batch scripts only work on Windows!

restart-server.bat

This batch file makes it easy to restart your server. Can also be used with the sys.system server-side scripting function.

Must be placed in the same directory as your server

This batch file makes the server run the server headless (no GUI).
