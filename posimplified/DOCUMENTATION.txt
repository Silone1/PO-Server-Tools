Pokemon Online Simplified Function Documentation

Module Documentation:

All modules should be created at the end of their file.
Some have args which you can change by editing the file.

Module classes should be prefixed with _ and should not have that when loaded.
Example:
_POSomething = function() {}
POSomething = new _POSomething(args);

Functions should be defined with .prototype:
POSomething.prototype.something = function(args)

They should be called (outside of the file) without prototype., and inside with this.{function}

Modules can exist a number of times in multiple variables.
Example:
var posomthing = new _POSomething(args);
POSOM = new _POSomething(args);
POSOMSC = POSomething;


This allows scripts to use modules without having to request the user to edit the module file.
You SHOULD ALSO NOT EVER delete a module's default function (for example, delete POCache), as other modules might require it!

You will find information for a Custom Module in it's file.

===== PO. functions =====

All Functions made by TheUnknownOne unless specified.
core.js:

void PO(string path, string ini_file)
     Inits PO Simplified and loads all modules.
	 (string) path = path for ini file.
	 (string) ini_file = ini file in directory specified by path.
         Notes: Is not a member of PO.

string testPath(string path)
       Puts a slash to the path if needed.
	   (string) path = path to test.
	   (string) return = formatted path.
	   Notes: Created when void PO is called.

mix (void/string) includeModule(string module_path, string module_file,
                                (optional) string _name, (optional) boolean _function)
		Includes a module with PO Simplified.
        (string) module_path = path to module.
        (string) module_file = file in directory specified by path.
        (optional) (string) _name = name for module.
        (optional) (boolean) _function = return function in string & add in module
         information array.
        (mix) (void/string) return = (void) / (string) = returns function.
        Notes: Created when void PO is called.

mix (string/any_but_void) ini_set(string setting, mix (any_but_void) value)
        Changes an ini setting (does not change in file).
        (string) setting = ini setting name.
        (mix) (any_but_void) value = new value for setting. type must be the same.
        (mix) (string/any_but_void) return = (string) = error message / (any_but_void) =
         old setting.
		 
mix (string/object) ini_restore()
        Restores ini to the one loaded at creation.
        (mix) (string/object) return = (string) = error message / (object) =
         old ini.
		 
misc.js:
 
mix (any/string) callsysfunc(string function[, mix (any) arguments])
         Calls a sys function with given arguments. Each argument should be seperated
         by a new parameter.
         (string) function = the function to run.
         (mix) (any/string) return = (any) = result from function call / (string) =
          error message.
